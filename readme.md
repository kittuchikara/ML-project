Feature importance analysis shows that both textual and contextual signals contribute to emotional state prediction.
Metadata features such as duration_min, sleep_hours, stress_level, and energy_level were among the most important predictors, indicating that physiological and situational context strongly influences emotional state.

Textual features extracted using TF-IDF also played a key role. Words such as “felt”, “lighter”, “calmer”, and “drained” were important indicators of emotional expression within user reflections.

This confirms that combining text + metadata improves emotional understanding compared to using text alone.
